* runemacs.bat
最开始的方案是在bat文件里指定HOME目录
%~dp0: %0代表脚本自身，~dp表示变量扩充，d是扩充到分区，p是扩充到路径。%~dp0合起来就代表当前目录的路径。
set HOME=%~dp0
  #+begin_src
set HOME=C:\Users\<username>
"C:\portable\emacs-27.2-x86_64\bin\runemacs.exe"
  #+end_src
